- 희망 분야와 관련하여 활동한 내용, 공부한 내용 등에 대해 기재해 주세요.

Javascript라는 스크립트 언어가 가지는 유연함과 간결한 스타일의 개발 방식이 저의 개발 적성에 잘 맞는 것 같아 프론트엔드 분야에 관심을 가지게 되었습니다.
그래서 현재 코드스쿼드라는 교육기관에서 좋은 마스터분들과 동료들을 만나 학습하며 성장해 나가고 있습니다.
처음 자바스크립트를 배웠을 때는 예전에 배운 C, C++, Java같은 절차적 프로그래밍 언어의 방식과 달라 혼동이 오기도 했지만
자바스크립트가 가지는 비교적 간단한 문법들과 유연한 코딩 방식덕분에 빠르게 배워 나갈 수 있었습니다.
절차형 형태의 코딩 스타일로 개발하던 초기와는 다르게 자바스크립트 특유의 객체를 배운 후에는 객체지향형 스타일로 개발을 하고 있습니다.
HTML5, CSS, Javascript를 사용해 기초적인 동적 웹페이지나 UI들을 만들어 보며 공부했는데
그중 뉴스스탠드를 MVC 패턴으로 개발해본게 가장 연습이 많이 된거 같습니다.(https://github.com/latilt/NewsStand)
개발에 필요한 기술들은 MDN이나 라이브러리 개발자가 지원하는 공식 문서들을 찾아보며
궁금한 점들은 google이나 stack overflow 사이트를 활용하여 답을 찾아보고 합니다.
React 라이브러리를 사용해 프로젝트를 진행중이며 Github를 사용해 코드 관리와 이슈 기반의 협업 활동을 하고 있습니다.
Github 주소 : https://github.com/latilt

1) 진행기간 : 2017.04.17 ~ 2017.05.31
2) 프로젝트명 : codedu, 초급 개발자들을 위한 개발 언어 학습 컨텐츠를 제공하는 웹사이트. 주제별로 학습 컨텐츠를 제공하며 문제 풀이를 통해 학습 결과를 확인할 수 있다.
3) Backlog 작성(공동), 로그인 서비스, Header 컴포넌트, 학습내용소개(Study) 컴포넌트, 문제 풀이(Quiz) 컴포넌트, 퀴즈완료화면(QuizResult) 컴포넌트, 해당 컴포넌트들에 필요한 서버 API 개발을 담당하여 프로젝트를 진행.
4) 프로젝트 1주차에는 주제 선정, Mockup 디자인, Backlog 작성, 데이터베이스 설계, 팀 룰 설정 등의 기본 작업을 한 후 2주차부터 개발을 진행.
주중 오전에는 마스터들의 강의를 학습하고 오후부터 프로젝트 개발에 시간을 할애함.
5) 프론트엔드는 React와 Semantic UI를 백엔드는 NodeJs를 데이터베이스는 MySql과 MongoDB를 사용함.
6) 깃허브의 이슈관리와 풀리퀘스트 통한 팀원과의 협업 경험, 개별 브랜치를 나눠 개발하며 풀리퀘스트를 통한 머지로 코드 관리 경험,
하나의 페이지를 구성하는 주요 컴포넌트와 서버 API를 묶어 한 사람이 전담하여 개발함으로써 프론트엔드에서 백엔드, 데이터베이스까지 한 싸이클을 골고루 학습하며 개발할 수 있었음. 프레임워크와 라이브러리를 사용해 봄으로써 얻는 개발간에 얻을 수 있는 이점들(개발 시간 단축, 개발 편의성 제공) 경험.

능숙한 언어
Javascript : 현재 가장 주력으로 배우며 사용하는 언어입니다. 네이티브 자바스크립트만 사용해 2번의 연습용 미니프로젝트를 진행해봤습니다.
현재 진행중인 프로젝트는 React 라이브러리를 사용하여 개발중이며 이를 통해 ES6문법을 학습하고 있습니다.
프로젝트에서는 자바스크립트 기반의 서버인 NodeJs를 주로 사용하고 있습니다.

사용가능언어
HTML5, CSS : 미니프로젝트 기간동안 자바스크립트와 같이 프론트엔드 개발에 주로 사용했으며 React의 Component와 Semantic UI를 사용해 페이지를 구성해 보기도 했습니다.
Java : 알고리즘이나 자료구조에 대한 학습에 주로 사용하고 있습니다.
Mysql : 관계형 데이터베이스의 기본으로 배우고 있으며, 주로 로그인, 회원가입의 사용자 영역에 대한 데이터를 저장하는데 사용하고 있습니다.
MongoDb : NodeJs를 사용하는데 가장 알맞는 데이터베이스로 알려져 있어 학습하고 있으며, 현재 프로젝트의 학습내용과 문제 영역의 데이터를 보관하고 가져오는데 관계형 데이터베이스보다 간편하며 빠르다는 이점이 있어 사용중입니다.
